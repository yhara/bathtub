
バ ス タ ブ 一 直 線

 C67 バージョン

【前口上】
昨年のC65にてEndEffectorの「おまけ」として収録されていたバスタブ一直線が、
１年の時を経てここに復活！進化したようなそうでもないような雰囲気をお楽しみください。

【遊び方】
放物線を描いて飛んでくる人々を、バスタブでキャッチしてください。
３回ミスるとゲームオーバーです。

 EASY: 簡単です。
 NORMAL: ふつうです。
 ENDLESS: 「終わりがない」と書いてエンドレスです。飽きたらESCキーで終了してください。 :-P

【操作】
左右キーで移動。
もしくは、スペースorエンターキーで方向転換。

ネームエントリーは左右で移動、スペースキーで決定。（実はキーボードも使えます）

【謝辞】
バスタブ一直線はRuby/SDLを用いて製作され、exerbを用いて実行ファイルを
作りました。
- オブジェクト指向スクリプト言語Ruby: http://www.ruby-lang.org/ja/
- SDL(Simple DirectMedia Layer): http://www.libsdl.org/
- Ruby/SDL: http://www.kmc.gr.jp/~ohai/rubysdl.html
- exerb: http://exerb.sourceforge.jp/

またバスタブに飛び込む効果音はWEB WAVE LIB様のものを使わせてもらってます。
- WEB WAVE LIB: http://www.s-t-t.com/wwl/
